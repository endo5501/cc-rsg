# Context optimisation mode B

Mode B keeps chapter bodies out of the main conversation by relaying a
manifest, path, status, summary, and question IDs.

It is available only for explicitly selected parallel execution on a runtime
that supports delegation. The authoritative workflow remains:

- `../../references/phase-3-investigate.md`
- `../../references/chapter-investigator-prompt.md`
- `SKILL.phase3-stepG.md`

Sequential execution remains the default and does not use Mode B.
